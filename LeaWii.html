LiiWii associates functions with buttons on a wiimote



Creation / Class Methods


*new (netAddr)

Instantiates a List to contain the osc responders, and adds a function to CmdPeriod to subsequently free these responders.

netAddr defaults to 127.0.0.1:8000, which is internal loopback and port for osculator

Instance Variables

add (item, func, device number, net address)

Item is an abbreviation of the wiimote control name i.e. a for button a etc. mostly self evident, otherwise look at class file.

func is a function to be associated with the controller

device number is the wiimote device number, again this assumes the osculator software which allows for up to four wiimotes to be controlled
